# 🌐 What is Web Development?

Web Development is the process of building websites and web applications that run on the internet.

It includes:
- Creating structure (HTML)
- Styling design (CSS)
- Adding interactivity (JavaScript)
- Managing servers & databases (Backend)

---

## 🧠 Types of Web Development

Web development is divided into three main parts:

### 1️⃣ Frontend Development
Frontend is what users see and interact with.

Technologies:
- HTML
- CSS
- JavaScript
- React

![Frontend Architecture](/assets/webdev/frontend-diagram.png)

---

### 2️⃣ Backend Development
Backend handles:
- Server
- Database
- Authentication
- API logic

Technologies:
- Node.js
- Express
- MongoDB

![Backend Flow](/assets/webdev/backend-flow.png)

---

### 3️⃣ Full Stack Development
Full Stack = Frontend + Backend + Database

A Full Stack Developer builds complete applications.

---

## 🔄 How the Web Works

User → Browser → Server → Database → Server → Browser

![Client Server Model](/assets/webdev/client-server.png)

---

## 🚀 Real World Example

When you open a website:

1. Browser sends request to server
2. Server processes logic
3. Server sends response
4. Browser renders UI
